The Problem

Configuration Drift is Invisible

🕐
Days of Lag
Manual configuration reviews take days to weeks. Attackers exploit misconfigurations within hours.
🔔
Alert Fatigue
Security teams receive alerts with no context, cause, or remediation steps — leading to ignored alerts.
📋
Compliance Blind Spots
Drifts are not automatically mapped to NIST, GDPR, or PCI-DSS, making audit reporting manual and error-prone.
👤
No Operator Accountability
Who made the change? When? Why? There is no risk ranking of operators making high-risk changes.
🌙
Off-Hours Blind Spots
Security changes at 2AM are especially dangerous — no automated detection of suspicious timing windows.
🔄
Silent Regressions
Controls silently disabled after being enabled — the most dangerous drift pattern — go undetected.

Algorithm

6-Factor Risk Scoring Model

# Multi-Factor Additive Scoring risk_score = max(0, severity_weight # 0–40 + status_weight # −10 to +30 + change_type_weight # −5 to +25 + regression_bonus # severity-gated + off_hours_bonus # +10 if 00:00–07:00 + compliance_bonus # +10 if mapped ) # Severity ceiling applied AFTER scoring Info → max BENIGN Low → max MEDIUM_DRIFT Medium → max MEDIUM_DRIFT # key FPR fix High → max HIGH_DRIFT Critical → CRITICAL_DRIFT
Score Breakdown Factors
Severity +0 to +40
Status −10 to +30
Change Type (Disable) +25
Regression Bonus +10 / +20 (sev-gated)
Off-Hours +10
Compliance Present +10
Thresholds: CRITICAL ≥ 70 · HIGH 50–69 · MEDIUM 30–49 · BENIGN <30
Calibration

False Positive Rate: 50% → 5.4%

v1 Baseline
FPR = 50.4% · Recall = 95.2%
Low/Info events getting +20 regression bonus regardless of severity → promoted to CRITICAL. Fixed-weight regression bonus was the root cause.
v2 Fix 1
FPR = 25.0% · Recall = 94.2%
Severity-gated regression bonus introduced: Low/Info → 0 bonus, Medium → +10. Compliant/Remediated status gets −5 penalty. FPR halved.
v3 Final
FPR = 5.4% · Recall = 93.9% · F1 = 91.3%
Medium severity capped at MEDIUM_DRIFT. Compliant/Remediated penalty increased to −10. 37 FPs remain from inherently ambiguous edge cases.
